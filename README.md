# Sri Puja Store — UI/UX & Full Stack Redesign

Intern assignment redesigning sripujastoreusa.com — 
an Indian Hindu puja products ecommerce store serving 
the Indian diaspora in the USA.

## Project Overview
A complete UI/UX redesign and full stack development 
plan for Sri Puja Store, improving the mobile shopping 
experience, product discovery, and checkout flow.

## Screens Designed
- Home Page
- Product Listing Page
- Product Detail Page
- Checkout & Payment Page

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Next.js + React |
| Styling | Tailwind CSS |
| Backend | Node.js + Express |
| Database | Supabase (PostgreSQL) |
| Authentication | Supabase Auth |
| Hosting | Vercel |
| CMS | Headless WordPress |

UI/UX design complete — development phase upcoming
